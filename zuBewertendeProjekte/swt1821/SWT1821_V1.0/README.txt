Da es leider zu einigen Problemen beim login kommtmuss die Applikation über elipse gestartet werden.
Hiefür ist es von Nöten einigejars einzubinden, die im Ordenr "libs" zufinden sind. Klicken SIe hierfür auf den obersten Ordener
des Projektes und nun auf File->Properties->Java Build Path->Add JARs... und fügen sie alle .jar Dateien aus dem Ordner "libs" hinzu.

Das Programm lässt sich starten indem man die Klassen "VisualClass" oder "NeuenGruppenraumErstellen" aus dem package "gui" als Java Application ausführt.
